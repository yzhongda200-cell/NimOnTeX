# NimOnTeX

A Nim game engine implemented entirely in TeX macro expansion system.

---

## Overview

NimOnTeX is a fully TeX-based implementation of the game Nim.

It behaves as a small interactive system with:

- Game logic
- State transition system
- Turn-based interaction
- History tracking

All behavior is realized within TeX macro expansion rules.

---

## Features

- Nim gameplay inside TeX
- Turn-based interaction (You vs TeX)
- Automatic move computation
